# Cryptographic Chat Application

A simple encrypted chat application written in Python.

## Features

- End-to-end symmetric encryption using Fernet (AES-based)
- TCP socket communication
- Local network demonstration
- Educational project

## Installation

```bash
pip install -r requirements.txt
```

Generate a key:

```bash
python generate_key.py
```

Start the server:

```bash
python server.py
```

Start the client:

```bash
python client.py
```
